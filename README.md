## Technologies
- React.js (Functionality)
- THREE.js (Landing page 3D rendering)
- react-spring (Animations)
- react-typist (Typewriter animations)
- Bootstrap (Design)


## How To Run Locally
Using `npm`:
1. Change directory to `client`
```bash
cd client
```
2. Install dependencies
```bash
npm install
```
3. Run locally
```bash
npm start
```
