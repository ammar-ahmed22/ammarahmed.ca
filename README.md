## Tech Stack

**Client:** React, Apollo, Chakra-UI

**Server:** Node, Express, GraphQL, Notion API

**Database:** Notion


## Features

- Light/dark mode toggle
- Projects, blogs and work experience data persisted in Notion
- Add new information to site right from Notion
